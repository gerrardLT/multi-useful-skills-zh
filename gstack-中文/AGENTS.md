# gstack — 人工智能工程工作流程

gstack 是 SKILL.md 文件的集合，为 AI 代理提供结构化角色
软件开发。每项技能都是专家：CEO 审核员、工程经理、
设计师、QA 主管、发布工程师、调试员等。

## 构建命令

```bash
bun install              # install dependencies
bun test                 # run tests (free, <5s)
bun run build            # generate docs + compile binaries
bun run gen:skill-docs   # regenerate SKILL.md files from templates
bun run skill:check      # health dashboard for all skills
```

## 主要约定

- SKILL.md 文件是从 `.tmpl` 模板**生成**的。编辑模板，而不是输出。
- 运行 `bun run gen:skill-docs --host codex` 以重新生成 Codex 特定的输出。
- 浏览二进制文件提供无头浏览器访问。在技​​能中使用`$B <command>`。
- 安全技能（小心、冻结、防护）使用内嵌的咨询性散文——在破坏性操作之前始终进行确认。
