# 设计系统——gstack

## 产品背景
- **这是什么：** gstack 社区网站 — 一个 CLI 工具，可将 Claude Code 转变为虚拟工程团队
- **它的对象：** 发现 gstack 的开发人员、现有社区成员
- **Space/industry:** 开发者工具（同级：Linear、Raycast、Warp、Zed）
- **项目类型：** 社区仪表板 + 营销网站

## 审美方向
- **方向：** Industrial/Utilitarian — 功能优先、数据密集、等宽字体作为个性字体
- **装饰级别：** 故意的 - 表面上的微妙噪音/grain纹理以实现物质性
- **心情：** 由关心工艺的人打造的严肃工具。温暖，不冷。 CLI 的传承就是品牌。
- **参考站点：** Formulae.brew.sh（竞争对手，但我们是实时和交互式的），Linear（黑暗+克制），Warp（温暖的口音）

## 版式
- **Display/Hero:** Satoshi (Black 900 / Bold 700) — 温暖的几何图形，独特的字母形式（小写“a”和“g”）。不是国际米兰，不是盖斯特。从 Fontshare CDN 加载。
- **正文：** DM Sans（常规 400 / 中号 500 / 半粗体 600） — 干净、可读，比几何显示稍微友好。从 Google 字体加载。
- **UI/Labels:** DM Sans（与正文相同）
- **Data/Tables:** JetBrains Mono (Regular 400 / Medium 500) — 个性字体。支持表格数字。等宽字体应该突出，而不是隐藏在代码块中。从 Google 字体加载。
- **代码：** JetBrains Mono
- **正在加载：** DM Sans + JetBrains Mono 的 Google 字体，Satoshi 的 Fontshare。使用 `display=swap`。
- **规模：**
- 英雄：72px / 夹子（40px、6vw、72px）
- H1：48 像素
- H2：32 像素
- H3：24 像素
- H4：18 像素
- 主体：16 像素
- 小：14 像素
- 标题：13 像素
- 微型：12 像素
- Nano：11px（JetBrains Mono 标签）

## 颜色
- **方法：** 克制 — 琥珀色口音很少见且有意义。仪表板数据获取颜色；铬保持中性。
- **主要（深色模式）：** amber-500 #F59E0B — 温暖、充满活力，读作“终端光标”
- **主要（浅色模式）：** amber-600 #D97706 — 较暗，与白色背景形成对比
- **主要文本重音（深色模式）：** amber-400 #FBBF24
- **主要文本重音（浅色模式）：** amber-700 #B45309
- **中性色：** 冷锌灰色
- 锌 50：#FAFAFA（最轻）
- 锌-400：#A1A1AA
- 锌 600：#52525B
- 锌-800：#27272A
- 表面（深色）：#141414
- 底座（深色）：#0C0C0C
- 表面（光）：#FFFFFF
- 底座（光）：#FAFAF9
- **语义：**成功#22C55E，警告#F59E0B，错误#EF4444，信息#3B82F6
- **深色模式：** 默认。近乎黑色的底座 (#0C0C0C)，表面卡片为#141414，边框为#262626。
- **灯光模式：** 暖色石材底座 (#FAFAF9)、白色表面卡片、石材边框 (#E7E5E4)。琥珀色重音变为 amber-600 以形成对比。

## 间距
- **基本单位：** 4px
- **密度：** 舒适 - 不拥挤（不是彭博终端），不宽敞（不是营销网站）
- **比例：** 2xs(2px) xs(4px) sm(8px) md(16px) lg(24px) xl(32px) 2xl(48px) 3xl(64px)

## 布局
- **方法：** 仪表板采用网格规则，登陆页面采用编辑英雄
- **网格：** lg+ 12 列，移动设备 1 列
- **最大内容宽度：** 1200px (6xl)
- **边框半径：** sm：4px，md：8px，lg：12px，full：9999px
- 卡片/panels：lg (12px)
- 按钮/inputs: md (8px)
- 徽章/pills：完整（9999px）
- 技能栏：sm (4px)

## 运动
- **方法：** 最小功能 - 仅有助于理解的过渡。仪表板的实时反馈就是动作。
- **缓动：** 输入（缓出 / 立方贝塞尔曲线（0.16,1,0.3,1）） 退出（缓入） 移动（缓入-出）
- **持续时间：** 微（50-100ms）短（150ms）中（250ms）长（400ms）
- **动画元素：**实时馈送点脉冲（2秒无限）、技能栏填充（600毫秒缓出）、悬停状态（150毫秒）

## 纹理
将微妙的噪音叠加到整个页面以增加重要性：
- 深色模式：不透明度 0.03
- 灯光模式：不透明度0.02
- 使用 SVG feTurbulence 滤镜作为 body::after 上的 CSS 背景图像
- 指针事件：无，位置：固定，z-index：9999
