# QA 问题分类

## 严重程度

|严重性|定义|示例|
|----------|------------|----------|
|**批判的**|阻止核心工作流程、导致数据丢失或应用程序崩溃|表单提交导致错误页面、结账流程中断、数据未经确认即被删除|
|**高的**|主要功能损坏或无法使用，没有解决方法|搜索返回错误结果、文件上传无提示失败、身份验证重定向循环|
|**中等的**|功能有效，但存在明显问题，存在解决方法|页面加载缓慢（>5 秒），缺少表单验证但提交仍然有效，布局仅在移动设备上损坏|
|**低的**|轻微的外观或抛光问题|页脚中的拼写错误、1px 对齐问题、悬停状态不一致|

## 类别

### 1.视觉/UI
- 布局中断（重叠元素、剪切文本、水平滚动条）
- 图像损坏或丢失
- z-index 不正确（元素出现在其他元素后面）
- 字体/color 不一致
- 动画故障（卡顿、过渡不完整）
- 对齐问题（离网、间距不均匀）
- 深色模式/主题问题

### 2. 功能性
- 链接损坏（404，目的地错误）
- 死按钮（点击不执行任何操作）
- 表单验证（缺失、错误、绕过）
- 不正确的重定向
- 状态不持久（刷新、后退按钮时数据丢失）
- 竞争条件（双重提交、陈旧数据）
- 搜索返回错误或没有结果

### 3. 用户体验
- 令人困惑的导航（没有面包屑，死胡同）
- 缺少加载指示器（用户不知道发生了什么）
- 交互缓慢（>500ms，无反馈）
- 不清楚的错误消息（“出了问题”，没有详细信息）
- 破坏性行为前无需确认
- 跨页面交互模式不一致
- 死胡同（没有回头路，没有下一步行动）

### 4. 内容
- 拼写错误和语法错误
- 过时或不正确的文本
- 保留的占位符/lorem ipsum 文本
- 截断的文本（没有省略号或“更多”的情况下被截断）
- 按钮或表单字段上的标签错误
- 缺失或无用的空状态

### 5. 性能
- 页面加载缓慢（>3 秒）
- Janky 滚动（丢帧）
- 布局变化（加载后内容跳跃）
- 网络请求过多（单个页面>50）
- 未优化的大图像
- 阻止 JavaScript（页面在加载期间无响应）

### 6.控制台/Errors
- JavaScript 异常（未捕获的错误）
- 失败的网络请求（4xx、5xx）
- 弃用警告（即将发生的损坏）
- CORS错误
- 混合内容警告（HTTPS 上的 HTTP 资源）
- CSP 违规行为

### 7. 辅助功能
- 图像上缺少替代文本
- 未标记的表单输入
- 键盘导航损坏（无法使用 Tab 键切换到元素）
- 焦点陷阱（无法逃脱模式或下拉菜单）
- ARIA 属性缺失或不正确
- 色彩对比度不足
- 屏幕阅读器无法访问内容

## 每页探索清单

对于 QA 会话期间访问的每个页面：

1. **视觉扫描** — 获取带注释的屏幕截图 (`snapshot -i -a -o`)。查找布局问题、损坏的图像、对齐情况。
2. **交互式元素** — 单击每个按钮、链接和控件。每个人都按其所说的去做吗？
3. **表格** — 填写并提交。测试空提交、无效数据、边缘情况（长文本、特殊字符）。
4. **导航** — 检查 /out 中的所有路径。面包屑、后退按钮、深层链接、移动菜单。
5. **状态** — 检查空状态、加载状态、错误状态、full/overflow 状态。
6. **控制台** — 交互后运行 `console --errors`。是否有新的 JS 错误或失败的请求？
7. **响应能力** — 如果相关，请检查手机和平板电脑视口。
8. **身份验证边界** — 注销时会发生什么？不同的用户角色？
