## 如何新增一条公告

1. 创建 `docs/announcements/YYYY-MM-DD-<channel>-<topic>.md`
2. 包含：
   - post metadata（title、subtitle、opening）
   - 短版宣传帖
   - 长文 / 草稿（如适用）
3. 把文件加入本索引中的 `Latest` 和 `Timeline`
4. 在 `README.md` 顶部的 `Updates & Announcements` 区块中新增或更新链接
