# 给 PM 的平台指南

本指南旨在帮助您根据自身情况，选择最便捷的配置路径。

这是 v0.65 onboarding 更新的一部分：为 PM 团队提供更清晰的安装与集成路径。

如果您是非技术用户，请从这里开始：
- [`Using PM Skills 101.md`](Using%20PM%20Skills%20101.md)

## 快速选择

- “我想要最简单的聊天配置” -> [`Claude Desktop`](Using%20PM%20Skills%20with%20Claude%20Desktop.md) 或 [`ChatGPT Desktop`](Using%20PM%20Skills%20with%20ChatGPT%20Desktop.md)
- “我可以使用终端命令” -> [`Claude Code`](Using%20PM%20Skills%20with%20Claude%20Code.md)、[`Claude /slash commands`](Using%20PM%20Skills%20with%20Slash%20Commands%20101.md) 或 [`Codex`](Using%20PM%20Skills%20with%20Codex.md)
- “我需要自动化” -> [`n8n`](Using%20PM%20Skills%20with%20n8n.md) 或 [`LangFlow`](Using%20PM%20Skills%20with%20LangFlow.md)
- “我在测试其他技术栈” -> [`OpenClaw`](Using%20PM%20Skills%20with%20OpenClaw.md)
- “我在用 Anthropic 的 workspace 工具” -> [`Claude Cowork`](Using%20PM%20Skills%20with%20Claude%20Cowork.md)
- “我的团队在构建自定义 agents” -> [`Python agents`](Using%20PM%20Skills%20with%20Python%20Agents.md)
- “我在用 AI coding copilots” -> [`Cursor`](Using%20PM%20Skills%20with%20Cursor.md) 或 [`Windsurf`](Using%20PM%20Skills%20with%20Windsurf.md)
- “我在用 app/prototype builders” -> [`Bolt`](Using%20PM%20Skills%20with%20Bolt.md) 或 [`Replit Agent`](Using%20PM%20Skills%20with%20Replit%20Agent.md)
- “我在用企业工作流搭建器” -> [`Make.com`](Using%20PM%20Skills%20with%20Make.com.md)
- “我在用 autonomous engineering agents” -> [`Devin`](Using%20PM%20Skills%20with%20Devin.md)
- “我在运行 multi-agent orchestration” -> [`CrewAI`](Using%20PM%20Skills%20with%20CrewAI.md)
- “我主要用 Google 生态” -> [`Gemini`](Using%20PM%20Skills%20with%20Gemini.md)
