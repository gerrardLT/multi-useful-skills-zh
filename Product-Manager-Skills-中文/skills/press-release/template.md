# 新闻稿模板

使用此模板起草一份 Working Backwards 风格的新闻稿。

## 来源
改编自 `https://github.com/deanpeters/product-manager-prompts` 仓库中的 `prompts/visionary-press-release.md`。

## 模板
```markdown
**标题：**
"[公司] 推出 [产品/功能名称]，旨在 [主要益处/目标]"

**发稿地：**
"[城市]，[省/州]，[国家]，[日期] -

**引言：**
今日，[公司类型] 公司 [公司名称] 宣布 [关键新闻]，这是一项 [简要描述]。该 [产品/功能] 将 [主要益处]，以解决 [关键客户问题]。

**问题段落：**
[描述客户问题及其影响。如果可能，请包含一个支持性数据点或洞察。]

**解决方案段落：**
[以成果为导向，描述产品如何解决问题]。"[以客户为中心的引述]，" [公司领导] 表示。

**补充细节：**
[添加支持性优势、集成或数据点。]

**公司简介：**
[公司名称] 成立于 [年份]，是一家 [公司类型]，以 [主要产品/服务] 而闻名。专注于 [使命/价值观]，[公司名称] 已 [取得的成就或里程碑]。

**行动号召：**
有关 [产品/功能] 的更多信息，请访问 [网站] 或联系 [媒体联系人姓名]，联系方式：[联系信息]。

**媒体联系人：**
[姓名]
职位：[职位]
电话：[电话]
邮箱：[邮箱]
```
