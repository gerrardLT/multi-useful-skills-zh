# 路线图模板

使用此模板来制定一份以结果为导向、具有优先顺序的战略路线图。

## 模板
```markdown
# 产品路线图

## 战略背景
- [业务目标 / OKRs]
- [客户问题]
- [限制条件 / 依赖项]

## 路线图 (当前 / 下一步 / 后续)

| 阶段 | 举措 | 预期成果 | 衡量指标 | 备注 |
|---|---|---|---|---|
| 当前 | [举措] | [预期成果] | [衡量指标] | [备注] |
| 下一步 | [举措] | [预期成果] | [衡量指标] | [备注] |
| 后续 | [举措] | [预期成果] | [衡量指标] | [备注] |

## 排序 (可选)
- Q1: [举措]
- Q2: [举措]
- Q3: [举措]

## 风险与依赖
- [风险 1]
- [依赖项 1]
```
