# 产品战略会议模板

使用此模板记录一次战略会议的输出结果。

## 模板
```markdown
# 产品战略会议总结

## 战略背景
- [公司目标 / OKRs]
- [市场背景 / 竞争格局]

## 目标客户
- [主要用户画像]
- [关键待办任务 / 需求]

## 问题界定
- [问题陈述]
- [为何是现在]

## 机会点
- [主要机会]

## 解决方案假设
- [史诗级假设]

## 优先级排序
- [使用的框架 + 最高优先级]

## 路线图 / 排期
- [现在 / 接下来 / 未来 或 季度规划]

## 成功指标
- [关键指标]

## 后续步骤
- [实验、探索、PRD、路线图规划]
```
