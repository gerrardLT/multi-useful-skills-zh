# Discovery Process 总结模板

使用此模板记录发现过程的产出与决策。

## 模板
```markdown
# 发现总结

## 问题界定
- [问题陈述]
- [为何是现在]

## 研究洞察
- [主要的客户痛点]
- [引述 / 证据]
- [行为模式]

## 用户画像 / 待完成工作 (可选)
- [主要用户画像]
- [关键待完成工作]

## 机会
- [机会 1]
- [机会 2]
- [机会 3]

## 解决方案假设
- [假设 1]
- [假设 2]

## 已进行的实验
- [实验 + 结果]

## 决策
- [继续 / 放弃 / 转向]

## 后续步骤
- [产品需求文档、路线图、进一步发现]
```
