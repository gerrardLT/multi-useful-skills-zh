# EOL 消息模板

使用此模板撰写带有同理心和清晰迁移指引的 end-of-life 公告。

## 来源
改编自 `https://github.com/deanpeters/product-manager-prompts` 仓库中的 `prompts/eol-for-a-product-message.md`。

## 模板
```markdown
## End-of-Life Announcement: [Product Name]

### 产品过渡叙述

**我们是：** [公司及其与产品的关系]
- [对客户的承诺]
- [产品演进]
- [未来愿景]

**公告：**
- [清晰的 EOL 声明及替代方案]

**原因：**
- [原因 1：以客户利益为中心]
- [原因 2]
- [原因 3]

**这对您意味着：**
- [对客户的影响及益处]

### 当前产品背景

**我们的产品** [名称]
- **是一款** [简要描述]
- **已为** [目标客户] 服务 [时长]
- **通过提供** [关键益处]

### 客户影响

**我们理解这可能会通过以下方式影响您：**
- [影响 1]
- [影响 2]
- [影响 3]

### 过渡解决方案

**针对** [目标客户]
- **目前使用** [旧产品] 的用户
- [替代产品]
- **是一款** [类别]
- **它** [专注于连续性和改进的益处]

### 差异化与连续性

- **如同** [旧产品]
- [替代产品]
- **提供** [关键益处的连续性]
- **同时亦提供** [新益处]

### 支持与后续步骤

**为确保平稳过渡，我们将：**
- [支持措施 1]
- [支持措施 2]
- [支持措施 3]

### 时间表
- [关键日期 1 + 里程碑]
- [关键日期 2 + 里程碑]
- [关键日期 3 + 里程碑]

### 行动号召
- [后续步骤]
- [联系信息]
```
