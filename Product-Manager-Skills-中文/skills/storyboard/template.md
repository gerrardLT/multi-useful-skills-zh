# 分镜脚本模板

使用此模板收集输入，并输出一个 6 帧分镜叙事。

## 来源
改编自 `https://github.com/deanpeters/product-manager-prompts` 仓库中的 `prompts/storyboard-storytelling-prompt.md`。

## 输入
1. 主角（姓名、年龄、角色、背景）
2. 问题 / 挑战
3. “Oh Crap” 时刻
4. 解决方案引入
5. “Aha” 时刻
6. 解决方案后的生活
7. 视觉风格或素材（可选）

## 输出模板
```markdown
## 生成的 6 帧故事线

**第一帧：介绍主角**
- [描述主角、场景和背景]

**第二帧：问题浮现**
- [描述挑战及其影响]

**第三帧：“Oh Crap” 时刻**
- [升级问题并制造紧迫感]

**第四帧：解决方案出现**
- [介绍解决方案及初步反应]

**第五帧：“Aha” 时刻**
- [展示突破点]

**第六帧：解决方案后的生活**
- [描述改善后的状态]

**可选视觉元素**
- [若未提供风格：使用粗马克笔、Sharpie 风格草图、极简单色]
```
