## 基准与质量检查

### 收入指标
- [ ] 总收入与净收入已明确标注
- [ ] 收入增长率 > 成本增长率
- [ ] ARPU/ARPA 按同期群跟踪（而非仅混合计算）
- [ ] 收入集中度：最大客户 <10%，前 10 大客户 <40%

### 留存指标
- [ ] 月流失率 <5%（理想值 <2%）
- [ ] 已比较收入流失与客户流失
- [ ] NRR >100%（理想值 >120%）
- [ ] 速动比率 >2（理想值 >4）

### 同期群分析
- [ ] 近期同期群表现与早期同期群持平或更好
- [ ] 跟踪收入留存，而非仅客户留存
- [ ] 按同期群分析扩展收入

## 危险信号

检查是否存在以下情况：

- [ ] 收入流失 > 客户流失（表明流失的是大客户）
- [ ] ARPU 上涨，但客户数量下降（可能仅是结构变化，而非真正改善）
- [ ] 新同期群的流失速度快于旧同期群（产品市场契合度下降）
- [ ] NRR <100%（表明在收缩，而非扩张）
- [ ] 速动比率 <2（典型的“漏桶”问题）
- [ ] 折扣 >20% 或退款 >10%（通常意味着定价/产品问题）
- [ ] 前 10 大客户收入占比 >50%（风险过高）
- [ ] 扩展收入 < MRR 总额的 10%（变现结构存在问题）
