# 机会解决方案树模板

使用此模板记录成果、机会、解决方案和实验。

## 模板
```markdown
## 期望成果
- [要推动的业务或产品指标]

## 机会（待解决问题）
1. [机会 1]
2. [机会 2]
3. [机会 3]

## 每个机会的解决方案
**机会 1 解决方案：**
- [解决方案 1]
- [解决方案 2]
- [解决方案 3]

**机会 2 解决方案：**
- [解决方案 1]
- [解决方案 2]
- [解决方案 3]

## 实验（针对每个解决方案）
- [针对解决方案 1 的实验]
- [针对解决方案 2 的实验]

## 概念验证选择
- **选定方案：** [解决方案]
- **理由：** [可行性、影响力、市场契合度]
```
