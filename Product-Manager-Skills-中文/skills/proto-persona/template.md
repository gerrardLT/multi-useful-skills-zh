# Proto-Persona 模板

使用此模板记录一个基于假设的用户画像档案。

## 来源
改编自 `https://github.com/deanpeters/product-manager-prompts` 仓库中的 `prompts/proto-persona-profile.md`。

## 模板
```markdown
### 姓名
- [头韵式姓名]

### 传记与人口统计
- [年龄范围]
- [地理位置]
- [社会身份]
- [线上活跃度]
- [休闲活动]
- [职业状态]

### 引言
- "[引言 1]"
- "[引言 2]"
- "[引言 3]"

### 痛点
- [痛点 1]
- [痛点 2]
- [痛点 3]

### 此人试图达成什么？
- [行为或结果 1]
- [行为或结果 2]
- [行为或结果 3]

### 目标
- [目标 1]
- [目标 2]
- [目标 3]

### 态度与影响
- **决策权：** [是/否 + 背景说明]
- **决策影响者：** [谁影响此人]
- **信念与态度：** [影响决策的信念/态度]
```
