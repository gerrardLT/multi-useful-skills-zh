# Lean UX Canvas 模板

使用此模板记录精益用户体验画布的 8 个关键部分。

## 模板
```markdown
## Lean UX Canvas

### 1. 业务问题
- [发生了什么变化？为何现在它成为了一个问题？]

### 2. 业务成果
- [可衡量的行为变化 / KPI 目标]

### 3. 用户
- [主要的用户画像或细分群体]

### 4. 用户成果与收益
- [用户的目标、收益及情感结果]

### 5. 解决方案
- [待探索的候选方案]

### 6. 假设
- 我们相信，如果 [用户] 通过 [解决方案] 获得 [收益]，那么 [业务成果] 将得以实现。

### 7. 最高风险假设
- [需要首先验证的关键点]

### 8. 最小工作量 / 实验
- [用于验证最高风险假设的最小可行实验]
```
