```
[使用财务模型]
```

### LTV:CAC 比率
```
LTV: $__________
CAC: $__________
LTV:CAC 比率 = LTV / CAC = __________:1
```

**基准：**
- [ ] >3:1（健康，可放心扩张）
- [ ] 1-3:1（边缘状态，扩张前先优化）
- [ ] <1:1（不可持续，立刻修正）

### 回收期
```

```

**基准：**
- [ ] <12 个月（优秀）
- [ ] 12-18 个月（可接受）
- [ ] >24 个月（值得警惕，可能陷入现金陷阱）

## 效率比率

### 40 法则
```
收入增长率 (同比): __________%
利润率 %: __________%

40 法则 = 增长率 + 利润率 = __________
```

**基准：**
- [ ] >40（增长和效率平衡健康）
- [ ] 25-40（可接受）
- [ ] <25（值得警惕，烧钱但增长不够）

### 魔法数字
```
本季度收入: $__________
上季度收入: $__________
收入增长 = $__________

上季度销售与营销支出: $__________

魔法数字 = (收入增长 × 4) / 上季度销售与营销支出
魔法数字 = __________
```

**基准：**
- [ ] >0.75（效率高，可以放心放大销售与营销）
- [ ] 0.5-0.75（可接受，扩张前先优化）
- [ ] <0.5（效率差，先修正 GTM 再花钱）

## 基准与质量检查

### 单位经济学
- [ ] 毛利率 >70%（SaaS 应该是高毛利）
- [ ] LTV:CAC >3:1（可持续增长的最低线）
- [ ] 回收期 <12 个月（现金效率高）
- [ ] 贡献利润率 >60%（扣除所有可变成本后）
- [ ] LTV 使用毛利率计算，而不仅仅是收入

### 资本效率
- [ ] 现金跑道 >12 个月（健康缓冲）
- [ ] 净烧钱额随时间下降（有走向盈利的路径）
- [ ] 运营支出增长慢于收入（正向经营杠杆）
- [ ] 行政管理支出 < 收入的 15%（保持低管理开销）

### 效率比率
- [ ] 40 法则 >40（平衡健康）
- [ ] 魔法数字 >0.75（GTM 效率高）
- [ ] 收入增长率 > 运营支出增长率（存在经营杠杆）

---

## 危险信号

检查是否出现以下任意情况：

- [ ] LTV:CAC <1.5:1（亏本买收入）
- [ ] 回收期 >24 个月（现金陷阱）
- [ ] 现金跑道 <6 个月（生存危机）
- [ ] 40 法则 <25（烧钱但没有相应增长）
- [ ] 魔法数字 <0.5（GTM 引擎坏了）
- [ ] 运营支出增长快于收入（负向杠杆）
- [ ] 毛利率 <60%（毛利有问题）
- [ ] CAC 上升但 LTV 持平/下降（单位经济学在恶化）
- [ ] LTV:CAC 看起来很好，但回收期很差（健康假象）
