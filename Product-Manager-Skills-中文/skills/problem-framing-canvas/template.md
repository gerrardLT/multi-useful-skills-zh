# 问题框定画布模板

使用此模板记录 MITRE 问题框定画布的输出。

## 模板
```markdown
## 向内审视
- **问题（症状）：** [问题是什么？]
- **为何未解决：** [为什么尚未解决？]
- **我们的角色/偏见：** [我们可能如何成为问题的一部分？]

## 向外观察
- **受影响群体：** [人物画像/细分群体]
- **何时/何地：** [情境]
- **影响：** [后果]
- **其他相关方/排除群体：** [相邻/被排除的群体]
- **受益方：** [若问题持续或解决，谁会受益]

## 重新定义
- **问题重述：** [换一种方式陈述]
- **我们如何能够：** [可操作的 HMW 问题]
```
