# PESTEL 分析模板

使用此模板分析政治、经济、社会、技术、环境和法律因素。

## 来源
改编自 `https://github.com/deanpeters/product-manager-prompts` 仓库中的 `prompts/pestel-analysis-prompt-template.md`。

## 模板
```markdown
## 概述
- **项目/产品名称：** [名称]
- **分析目的：** [目标]
- **分析师：** [姓名/团队]
- **日期：** [日期]
- **地理范围：** [地区]
- **时间范围：** [时间范围]

## 1. 政治因素
- **政府政策：** [影响]
- **政治稳定性：** [影响]
- **贸易法规：** [影响]
- **税收政策：** [影响]

## 2. 经济因素
- **经济增长：** [影响]
- **通货膨胀率：** [影响]
- **汇率：** [影响]
- **消费者支出：** [影响]

## 3. 社会因素
- **人口结构：** [影响]
- **文化趋势：** [影响]
- **生活方式变化：** [影响]
- **消费者态度：** [影响]

## 4. 技术因素
- **技术进步：** [影响]
- **研发活动：** [影响]
- **自动化：** [影响]
- **数字化转型：** [影响]

## 5. 环境因素
- **气候变化：** [影响]
- **可持续发展实践：** [影响]
- **资源稀缺：** [影响]
- **环境法规：** [影响]

## 6. 法律因素
- **合规要求：** [影响]
- **知识产权：** [影响]
- **劳动法：** [影响]
- **健康与安全法规：** [影响]
```
