# 命令目录

由 `scripts/generate-catalog.py` 自动生成。请勿手动编辑。

## /discover

- 描述：运行结构化发现流程，涵盖从问题界定到机会映射与验证规划的全过程。
- 参数示例：`<问题、机会或功能领域>`
- 路径：`commands\discover.md`
- 使用工具：
  - `discovery-process`
  - `problem-framing-canvas`
  - `discovery-interview-prep`
  - `opportunity-solution-tree`
  - `pol-probe-advisor`
- 输出结果：
  - 发现计划
  - 优先级假设
  - 验证实验待办列表

## /leadership-transition

- 描述：指导产品经理向总监、副总裁/CPO的过渡规划，包含角色适配诊断与入职指导。
- 参数示例：`<当前角色、目标角色及过渡场景>`
- 路径：`commands\leadership-transition.md`
- 使用工具：
  - `altitude-horizon-framework`
  - `director-readiness-advisor`
  - `vp-cpo-readiness-advisor`
  - `executive-onboarding-playbook`
- 输出结果：
  - 过渡诊断
  - 角色准备度计划
  - 30-60-90天领导力行动计划
